# feedmypixel.com

This is feedMyPixel's source code.

feedMyPixel.com is built using [Jessie](https://github.com/rassie/jessie), Vanilla JavaScript, OOCSS and PHP.

Despite being open sourced, all feedMyPixel code and content remain copyright of Ben Chidgey.

## To compile project:

+ Go to project Root
+ $ npm install
+ $ grunt
+ Complied source code can be found in dist folder
